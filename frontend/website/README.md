## 🚀 **Quick Start**

<div style="background: linear-gradient(45deg, #4facfe, #00f2fe); padding: 20px; border-radius: 15px; margin: 20px 0;">

### 📦 **Installation**

```bash
# Navigate to frontend directory
cd frontend/website

# Install dependencies
npm install

# Install React Router (if not already installed)
npm install react-router-dom
```

### 🚀 **Development**

```bash
# Start the development server
npm run dev
```

🌐 **Frontend will be available at**: `http://localhost:5173`

### 🏗️ **Building for Production**

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

## 🔧 **Configuration**

<div style="background: linear-gradient(45deg, #4facfe, #00f2fe); padding: 20px; border-radius: 15px; margin: 20px 0;">

### ⚙️ **Vite Configuration**

```javascript
// vite.config.js
export default defineConfig({
  plugins: [react()],
  server: {
    port: 5173,
    proxy: {
      '/api': 'http://localhost:5000'
    }
  }
})
```

### 🎨 **Tailwind Configuration**

```javascript
// tailwind.config.js
module.exports = {
  content: ["./src/**/*.{js,jsx,ts,tsx}"],
  theme: {
    extend: {
      colors: {
        lunar: {
          primary: '#667eea',
          secondary: '#764ba2',
          dark: '#1a1a2e'
        }
      }
    }
  },
  plugins: []
}
```

### 🔍 **ESLint Configuration**

```javascript
// eslint.config.js
module.exports = {
  extends: ['react-app', 'react-app/jest'],
  rules: {
    'react/react-in-jsx-scope': 'off'
  }
}
```

### 🌐 **Deployment Options**

- **Netlify**: Drag and drop deployment
- **Vercel**: Git-based deployment
- **GitHub Pages**: Free hosting
- **AWS S3**: Scalable hosting

#### 🐳 **Docker Deployment**

```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build
EXPOSE 80
CMD ["npm", "run", "preview", "--", "--host", "0.0.0.0", "--port", "80"]
```

### ⚙️ **Environment Configuration**

```bash
# Production environment variables
VITE_API_URL=https://api.lunalens.com
VITE_APP_NAME=LunaLens
VITE_APP_VERSION=1.0.0
```

- **Solution**: Check API URL configuration

- **Problem**: Login not working
- **Solution**: Verify backend is running

- **Solution**: Check Tailwind responsive classes
- **Check**: Mobile device testing
